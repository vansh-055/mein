# Tic-Tac-Toe in C

Simple 2-player Tic-Tac-Toe game implemented in C for the terminal.

## How to Run

```bash
gcc main.c -o tic_tac_toe
./tic_tac_toe
```

## Features

- 2-player mode
- Clean terminal interface
- Validates moves
- Detects winner or draw
